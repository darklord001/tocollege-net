To instal gwt-google-apis.jar, just run:
./install 1.5.0.build137

For installing your own version of this jar:
1) Copy the version of gwt-google-apis.jar that you want to this directory

2) Make the script runable
chmod a+x install

3) then run
./install MVN_VERSION

with the argument MVN_VERSION = 1.5.0.build137 (or whatever version number you'd like)
